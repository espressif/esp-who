# Human Face Recognition Example

### Interaction

#### ESP-S3-EYE

Physical buttons

| btn  | opertion         |
|------|------------------|
| play | recognize        |
| up   | enroll           |
| down | delete last feat |

#### ESP-S3-KORVO-2

Physical buttons

| btn  | opertion         |
|------|------------------|
| play | recognize        |
| vol+ | enroll           |
| vol- | delete last feat |

#### ESP32-P4-Function-EV-Board

Touch screen buttons

| btn       | opertion         |
|-----------|------------------|
| recognize | recognize        |
| enroll    | enroll           |
| delete    | delete last feat |
